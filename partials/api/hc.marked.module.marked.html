marked
directive in module hc.marked

Description

Compiles source test into HTML.

Usage

as element:
<marked
       marked="{expression}"
       [opts="{expression}"]>
</marked>
as attribute
<any marked="{expression}"
     [opts="{expression}"]>
   ...
</any>

Parameters

ParamTypeDetails
markedexpression

The source text to be compiled. If blank uses content as the source.

opts
(optional)
expression

Hash of options that override defaults.
